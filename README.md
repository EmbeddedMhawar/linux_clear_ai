A Linux-native AI noise reduction plugin using MossFormer2. A free, open-source alternative to Supertone.

## Overview

This project aims to provide high-quality, real-time noise reduction for Linux audio, leveraging the MossFormer2 model. It is designed to be a drop-in replacement for proprietary solutions, offering privacy and performance on Linux systems.

## Features (Planned)

-   Real-time noise suppression
-   Low latency
-   Linux-native (LV2/VST3 plugin)
-   Open-source and free

## Status

This project is currently under active development. Features and stability are subject to change.
